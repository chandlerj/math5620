\textbf{Chapter 1:} Finite Difference Approximation\\

\textit{Example:} $\frac{dA}{dt} = kA, A(0) = A_0$\\


$\frac{1}{A} dA = k dt \rightarrow ln|A| = kt + c_0$\\

$A(t) = e^{kt + c_0} = c_1 e^{kt} \rightarrow c_1 = A_0$\\

$\boxed{A(t) = A_0 e^{kt}}$\\

\textit{Example:} $\frac{dP}{dt} = \alpha P - \beta P^2, P(0) = P_0$\\

\textit{Example - spring mass system:} $\frac{d^2 y}{dt^2} = \alpha \frac{dy}{dt} + ky = f(t), y(0) = y_0, \frac{dy}{dt}(0) = v_0$\\

\textit{Example:} $y'' + \frac{P}{EI} y = 0, y(0) = 0, y(L) = 0$\\

This example has the solution: $\boxed{y(x) = c_1 \cos(\lambda x) + c_2 \sin(\lambda x)},\\ assuming $y(L) = c_2 \sin(\lambda L) = 0, c_2 = 0, sin(\lambda L) = 0, \lambda L = n \pi$.\\

This means $y_n(x) = c_2 \sin(\frac{\pi n}{L} x)$\\

\textit{Example - Heat equation:} $\frac{\partial u}{\partial t} = k \frac{\partial^2 u}{\partial x^2},\\ u(x,0) = f(x), u(o, t) = g_1(t), u(', l) = g_2(l)$\\

\textit{Example - Potential Equation:} $\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = \Delta u = f(x, y)$\\
This a BVP, 2nd order, partial diff eq.\\

\textit{Example - Conservation Laws:} $\frac{\partial u}{\partial t} + a \frac{\partial u}{\partial x} = 0$\\
$u(x, o) = f(x), x \in \R$
